# Components

Reusable UI should live here. The current application shell is still centralized in `src/App.jsx`, so this folder is reserved for safe incremental extraction.

Recommended groups:

- `dashboard/` for dashboard cards, stats, and company selectors.
- `interview/` for transcript, chat, code editor panels, voice controls, and round controls.
- `reports/` for feedback cards, charts, and report search.
- `learning/` for the planned AI Learning Agent UI.
