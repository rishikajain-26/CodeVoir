# CodeVoir Demo Script

Use this flow for a concise hackathon demo.

## 1. Opening pitch

CodeVoir is an AI career-preparation platform. It interviews a student, diagnoses weak areas, teaches those weak areas from trusted sources, and helps them prepare for real opportunities.

## 2. Start from dashboard

Show:

- available companies,
- available interview rounds,
- DSA, CS fundamentals, and project/behavioral options,
- Learning Agent entry point,
- opportunity recommendation entry point.

## 3. Run a short DSA interview

Recommended actions:

1. Select a company with DSA data.
2. Start a medium round.
3. Let the AI ask the first question.
4. Explain the approach verbally.
5. Run code in the editor.
6. Submit code and show testcase feedback.

## 4. Show AI behavior

Point out:

- voice input,
- AI online/offline status,
- transcript drawer,
- live code editor,
- adaptive follow-up questions.

## 5. End and show report

Show:

- score breakdown,
- strengths,
- weaknesses,
- timeline/transcript,
- recommendation outcome.

## 6. Show the AI Learning Agent

Open **Learning Agent** from the dashboard.

Demo flow:

1. Paste a documentation URL or upload a short PDF.
2. Show the indexing status.
3. Ask: “Explain this like I only know basic JavaScript.”
4. Show answer, confidence, and source citations.
5. Generate notes or flashcards.
6. Export Markdown.

Suggested talking point:

> This is not a generic chatbot. The agent reads the user-provided source, indexes it, retrieves relevant sections, and answers from that source.

## 7. Show opportunities

Open the opportunities page and show how resume/profile information can be used to recommend hackathons, internships, jobs, or competitions.

Optional next step to mention:

- Opportunity cards can call the Learning Agent's preparation endpoint to generate a pitch, missing skills, and a 3-day prep plan.

## 8. Suggested closing line

CodeVoir is not only a mock interviewer. It is a preparation loop: interview, diagnose, learn, and apply.
