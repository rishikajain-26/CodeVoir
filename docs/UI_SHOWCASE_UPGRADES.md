# UI Showcase Upgrades

This build adds the final visual polish layer for the CodeVoir AI Learning Agent.

## Added

- Premium agent mission/status bar
- Agent progress stepper for source reading, retrieval, output structuring, and visual rendering
- Full-screen Present Mode for generated outputs
- Collapsible compact evidence drawer
- Animated 3D flashcard deck with flip interaction
- Canvas-style mind map with visual branches and connectors
- Premium empty state with direct action shortcuts
- Animated readiness heatmap bars
- Cleaner source/evidence grouping for demo clarity

## Demo tips

1. Upload a PDF or paste a documentation URL.
2. Generate Flashcards and use the 3D flip deck.
3. Generate Mind Map and show the canvas branch view.
4. Generate Skill Heatmap and show animated readiness bars.
5. Open Present Mode to make the output look like a pitch/demo slide.
6. Keep the evidence drawer collapsed until a judge asks about sources.
