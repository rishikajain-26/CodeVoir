# Out-of-the-Box CodeVoir Features

These features are built to create a strong hackathon demo moment without relying on LinkedIn or any external social platform.

## 1. Source Mock Interview
Upload a PDF, URL, GitHub repo, or notes. CodeVoir generates a mock interview directly from the selected source material.

Demo line: "The platform does not just explain the document; it can interview you from it."

## 2. Weak Answer Rewriter
Paste a rough interview answer. CodeVoir scores it, explains what is weak, rewrites it into a confident answer, and gives practice tips.

Demo line: "This closes the feedback loop from weak response to improved response."

## 3. Skill Readiness Heatmap
The AI agent turns source material or interview context into a readiness dashboard with scores for skills like DSA, projects, communication, and system design.

Demo line: "Students can instantly see where they are interview-ready and where they are not."

## 4. 10-Second Evaluator View
CodeVoir simulates what a judge or interviewer understands in the first 10 seconds of seeing the project or candidate profile.

Demo line: "This helps teams sharpen their first impression before the actual evaluation."

## 5. Judge Demo Kit
The AI agent generates a complete demo script: opening, problem statement, live demo flow, wow moments, architecture explanation, impact, and closing line.

Demo line: "The system prepares not only the candidate, but the entire presentation."

## 6. Visual Learning Outputs
Notes, flashcards, mind maps, heatmaps, and mock interviews are rendered as polished visual cards rather than plain text.

Demo line: "The AI output is not a wall of text; it becomes usable study material."
